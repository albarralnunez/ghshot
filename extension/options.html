ghshot

Uploads images to GitHub user-attachments using your existing github.com login. Add one or more bridges — e.g. a local one on http://127.0.0.1:41330, and/or a remote one on another machine (http://<host-ip>:41330). The token comes from each bridge's ~/.config/ghshot/bridge-token (or ghshot-bridge --print-token).

Prefer a loopback bridge (127.0.0.1). A remote bridge over plain http:// sends the token and image bytes unencrypted — only use one on a trusted private network.
