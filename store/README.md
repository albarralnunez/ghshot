# Store assets

Assets for publishing the extension to the Chrome Web Store (and Firefox AMO).

## Files

- `listing.md` — name, summary, description, and permission justifications to paste into
  the dashboard.
- `screenshot-1.png` — **placeholder** (1280×800), generated by
  `python3 scripts/gen-assets.py`. Replace with real captures before publishing.

## Screenshots to capture (1280×800 or 640×400, PNG/JPEG)

1. **The extension Options page** — Bridge URL + Token fields with a green "ok vX" after
   *Test connection*.
2. **A PR/issue comment** showing an image that ghshot posted, rendering **inline** on a
   private repo.
3. (Optional) **The terminal** running `ghshot shot.png --pr 42` next to the resulting
   comment.

## Icon

`extension/icons/icon128.png` is the store icon (auto-generated). To regenerate all icon
sizes after changing the design in `scripts/gen-assets.py`:

```bash
python3 scripts/gen-assets.py
```

## Packaging

From the repo root:

```bash
make zip          # → dist/ghshot-extension-<version>.zip  (upload this)
```
